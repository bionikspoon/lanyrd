============
Installation
============

At the command line either via easy_install or pip:

.. code-block:: shell

    $ pip install lanyrd

.. code-block:: shell

    $ easy_install lanyrd

Or, if you have virtualenvwrapper installed::

    $ mkvirtualenv lanyrd
    $ pip install lanyrd

**Uninstall**

.. code-block:: shell

    $ pip uninstall lanyrd
